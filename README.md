# Examples

Install:

```
/home/me $ git clone https://github.com/naughie/dotfiles.git
/home/me $ cd dotfiles
/home/me/dotfiles $ ./tools/make-$(uname -m)-$(uname -s)

# e.g.
/home/me/dotfiles $ ./tools/make-x86_64-Linux
```

## Prerequisites

```
$ apt install -y git curl build-essential musl-dev
```
